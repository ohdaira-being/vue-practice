Vue.jsの学習で製作したToDoアプリです。

1,立ち上げるには。
cssファイル、htmlファイル、jsファイルの３つをプルする。
html内にcssファイルとjsファイルの相対パスを書き替える。

2，仕様
・「新しいタスクの追加」
タスク名と期限を入力し、追加ボタンを押すことでタスクリストに追加することができます。
※タスク名が空欄だと追加できません。
・「タスクリスト」
追加したタスクの状態・編集・削除ができます。
状態は、完了と作業中があり、絞り込み機能で分けて表示することができます。
編集は、押すとテキストボックスの表示と編集ボタンが完了ボタンに変わります。
編集が終わったら、完了ボタンを押すと編集した内容がタスク名と期限に反映されます。
削除ボタンは、[Ctrl]を押しながらクリックするとタスクを削除できます。
